TEST ENGINE:

When user starts:

- Randomly select 12 questions
- Enforce GLOBAL 15-minute countdown
- Auto-submit when timer hits zero

Track for EACH question:

1. Start Delay  
(time from render → first interaction)

2. Time Taken

3. Confidence Level  
(low / medium / high)

4. Answer Change Behavior  
- wrong_to_right  
- right_to_wrong  
- none  

5. Ideal Time Deviation  
(actual / ideal)

6. Guess Detection  
If time < 40% ideal AND confidence low.

7. Precision Ratio  
If numeric answer exists, measure distance from correct.

-------------------------------------

COMPUTE INTELLIGENCE METRICS AFTER SUBMISSION:

Calculate:

✔ Total Score  
✔ Accuracy by difficulty  
✔ Average time deviation  

Detect:

✔ Carelessness

✔ Decision Volatility  
stable / productive_switcher / self_saboteur

✔ Cognitive Start Speed

✔ Trap Sensitivity

✔ Guess Probability Rate

✔ Precision Ratio Classification  
execution errors vs conceptual gaps

Store all metrics BEFORE calling AI.
